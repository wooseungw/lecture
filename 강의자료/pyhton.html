1. 변수 선언과 기본 자료형

변수는 데이터를 저장하는 공간입니다. 파이썬은 변수 선언 시 자료형을 명시하지 않으며, 할당되는 값에 따라 동적으로 타입이 결정됩니다.

주요 기본 자료형

  • str: 문자열 (예: "안녕하세요")
  • int: 정수 (예: 10, -5)
  • float: 실수 (예: 3.14, -0.5)
  • bool: 불리언 (True 또는 False)
# 1. 기본 변수 선언 및 값 할당
name = "김철수"  # 문자열(str) 변수
age = 25        # 정수(int) 변수
height = 175.5   # 실수(float) 변수
is_student = True  # 불리언(bool) 변수

print(f"이름: {name}")
print(f"나이: {age}")

# 2. 변수 타입 확인
print(f"name의 타입: {type(name)}")
print(f"age의 타입: {type(age)}")

3. 자료형 변환

데이터의 자료형을 다른 자료형으로 바꾸는 것을 '형변환'이라고 합니다. `str()`, `int()`, `float()`, `bool()` 등의 내장 함수를 사용합니다.

형변환이 필요한 순간

사용자로부터 입력받은 값은 항상 문자열(`str`)입니다. 이를 숫자로 계산하려면 반드시 `int()`나 `float()`로 변환해야 합니다.

# 6. 실생활 예제 - 나이 계산

birth_year = int(birth_year_str)
age = current_year - birth_year

print(f"입력값: '{birth_year_str}' ({type(birth_year_str)})")

# 숫자 -> 문자열 변환

4. 산술 및 비교 연산자

산술 연산자는 기본적인 사칙연산을 수행하며, 비교 연산자는 두 값의 크기를 비교하여 `True` 또는 `False`를 반환합니다.

주요 산술 연산자

  • `//` : 나눗셈의 몫
  • `%` : 나눗셈의 나머지
  • `**` : 거듭제곱
# 1. 산술 연산자
a = 15
b = 4
print(f"{a} // {b} = {a // b}")  # 몫
print(f"{a} % {b} = {a % b}")   # 나머지
print(f"{a} ** {b} = {a ** b}") # 15의 4제곱

# 2. 비교 연산자
x = 10
y = 20
print(f"{x} == {y}: {x == y}") # 같음
print(f"{x} != {y}: {x != y}") # 다름
print(f"{x} < {y}: {x < y}")   # 작음

5. 논리 및 할당 연산자

논리 연산자는 여러 조건을 조합하거나 반전시킬 때 사용합니다. 할당 연산자는 변수에 값을 할당하거나 연산 결과를 누적시킬 때 사용됩니다.

논리 연산자 종류

  • `and`: 두 조건이 모두 참일 때 True
  • `or`: 두 조건 중 하나라도 참일 때 True
  • `not`: 조건의 결과를 반전 (True -> False)
# 3. 논리 연산자
age = 25
has_license = True
print(f"운전 가능: {age >= 18 and has_license}")

# 4. 할당 연산자
score = 80
print(f"초기 점수: {score}")
score += 10  # score = score + 10
print(f"10점 추가 후: {score}")
score -= 5   # score = score - 5
print(f"5점 감점 후: {score}")

6. if-else 조건문

`if`문은 특정 조건이 참(True)일 때만 코드 블록을 실행합니다. `else`를 사용하면 조건이 거짓(False)일 때 실행할 코드를 지정할 수 있습니다.

들여쓰기의 중요성

파이썬은 중괄호 `{}` 대신 들여쓰기(indentation)로 코드의 종속 관계를 구분합니다. `if`나 `else`에 속한 코드는 반드시 4칸(또는 탭 1개) 들여써야 합니다.

# 1. 기본 if 문
age = 20
if age >= 19:
    print(f"나이: {age}세, 성인입니다.")

# 2. if-else 문
score = 85
if score >= 60:
    print(f"점수: {score}점, 합격입니다!")
else:
    print(f"점수: {score}점, 불합격입니다.")

# 6. 짝수/홀수 판별
number = 15
if number % 2 == 0:
    print(f"{number}는 짝수입니다.")
else:
    print(f"{number}는 홀수입니다.")

7. 중첩 조건문과 elif

조건문 안에 또 다른 조건문을 넣어 더 복잡한 조건을 처리할 수 있습니다. 여러 조건을 순차적으로 검사할 때는 `elif` (else if)를 사용합니다.

if, elif, else의 흐름

`if` 조건부터 순서대로 확인하고, 참인 조건을 만나면 해당 코드 블록만 실행한 뒤 전체 조건문을 빠져나옵니다. 모든 `if`와 `elif`가 거짓일 때만 `else`가 실행됩니다.

# 로그인 로직 (중첩 if)
CORRECT_ID = "python_user"
CORRECT_PW = "12345"

user_id = input("아이디: ")
if user_id == CORRECT_ID:
    user_pw = input("비밀번호: ")
    if user_pw == CORRECT_PW:
        print("로그인 성공!")
    else:
        print("비밀번호가 틀렸습니다.")
else:
    print("존재하지 않는 아이디입니다.")

# 학점 계산 (elif)
score = 85
if score >= 90:
    grade = "A"
elif score >= 80:
    grade = "B"
elif score >= 70:
    grade = "C"
else:
    grade = "F"
print(f"학점: {grade}")

Python 기초 - 반복문

8. for 반복문

`for`문은 정해진 횟수만큼 또는 리스트, 문자열 등 순회 가능한(iterable) 객체의 모든 요소를 차례대로 방문하며 코드를 반복 실행합니다.

`range()` 함수

`range(start, stop)`는 `start`부터 `stop-1`까지의 숫자 시퀀스를 생성합니다. `for`문과 함께 사용하여 특정 횟수만큼 반복하는 데 매우 유용합니다.

# 구구단 출력 예제
dan = 7
if 2 <= dan <= 9:
    print(f"--- {dan}단 ---")
    # for 반복문: 1부터 9까지 순회
    for i in range(1, 10):
        print(f"{dan} x {i} = {dan * i}")
else:
    print("2~9 사이의 숫자만 입력해주세요.")

# 리스트 순회
fruits = ["사과", "바나나", "오렌지"]
for fruit in fruits:
    print(fruit)

9. while 반복문

`while`문은 주어진 조건이 참(True)인 동안 코드를 계속해서 반복 실행합니다. 주로 반복 횟수가 정해지지 않았을 때 사용됩니다.

무한 루프와 `break`

`while True:`는 의도적인 무한 루프를 만듭니다. 루프 내부에서 특정 조건을 만족했을 때 `break`를 사용하여 반복을 강제로 종료해야 합니다.

# 숫자 맞추기 게임
import random
secret_number = random.randint(1, 100)
attempts = 0

while True:
    guess = int(input("숫자를 맞춰보세요: "))
    attempts += 1

    if guess < secret_number:
        print("더 큰 숫자입니다!")
    elif guess > secret_number:
        print("더 작은 숫자입니다!")
    else:
        print(f"정답! {attempts}번 만에 맞췄습니다.")
        break # 루프 탈출

10. 리스트 생성과 요소 접근

리스트는 여러 개의 데이터를 순서대로 저장하는 자료구조입니다. 대괄호 `[]`를 사용하여 생성하며, 각 요소는 쉼표로 구분합니다.

인덱스(Index)

리스트의 각 요소는 0부터 시작하는 고유한 번호(인덱스)를 가집니다. 인덱스를 사용하여 특정 위치의 요소에 접근할 수 있습니다. 음수 인덱스는 뒤에서부터 접근합니다. (예: `-1`은 마지막 요소)

# 1. 리스트 생성
fruits = ["사과", "바나나", "오렌지"]
numbers = [1, 2, 3, 4, 5]
mixed = ["안녕", 123, True]

# 2. 리스트 요소 접근 (인덱스)

# 6. 리스트 슬라이싱
print(f"처음 3개 숫자: {numbers[0:3]}")

11. 리스트 조작과 정렬

리스트는 생성 후에도 요소를 추가, 수정, 삭제할 수 있는 '가변(mutable)' 자료형입니다. 다양한 내장 메서드를 제공합니다.

자주 쓰이는 리스트 메서드

  • `.append()`: 맨 뒤에 요소 추가

12. 딕셔너리 생성과 활용

딕셔너리는 '키(Key)'와 '값(Value)'을 하나의 쌍으로 묶어 저장하는 자료구조입니다. 중괄호 `{}`를 사용하여 생성하며, 순서가 없습니다.

키(Key)의 특징

키는 딕셔너리 내에서 고유해야 하며, 변하지 않는 값(주로 문자열이나 숫자)을 사용합니다. 키를 통해 해당 값을 빠르고 효율적으로 찾을 수 있습니다.

# 1. 딕셔너리 생성
student = {
    "이름": "김철수",
    "나이": 20,
    "학과": "컴퓨터공학"
}

# 2. 값 접근
print(f"학생 이름: {student['이름']}")

# 3. 새 항목 추가
student["성적"] = "A"
print(f"추가 후: {student}")

# 4. 값 수정
student["나이"] = 21
print(f"수정 후: {student}")

# 5. 항목 삭제
student.pop("학과")
print(f"삭제 후: {student}")

13. 딕셔너리 반복과 중첩

딕셔너리는 `for`문을 통해 키, 값, 또는 키-값 쌍을 순회할 수 있습니다. 딕셔너리의 값으로 또 다른 딕셔너리를 넣어 복잡한 데이터를 구조화할 수 있습니다.

`.items()` 메서드

`for key, value in student.items():` 와 같이 사용하면 딕셔너리의 키와 값을 동시에 가져와 반복문을 실행할 수 있어 매우 편리합니다.

# 7. 딕셔너리 반복문
student = {"이름": "김철수", "나이": 20, "학과": "컴퓨터공학"}

for key, value in student.items():
    print(f"  {key} → {value}")

# 8. 중첩 딕셔너리
students = {

}

# 이영희의 나이 접근
print(f"이영희의 나이: {students['이영희']['나이']}")

14. 튜플 생성과 언패킹

튜플은 리스트와 유사하지만, 생성된 후에 그 값을 변경할 수 없는 '불변(immutable)' 자료형입니다. 소괄호 `()`를 사용하여 생성합니다.

튜플 언패킹(Unpacking)

튜플의 각 요소를 여러 변수에 한 번에 할당하는 기능입니다. 함수의 다중 반환 값 처리나 변수 값 교환에 유용하게 사용됩니다.

# 1. 튜플 생성
coordinates = (10, 20)
colors = ("빨강", "초록", "파랑")

# 3. 튜플은 수정할 수 없음
# colors[0] = "노랑"  -> TypeError 발생

# 4. 튜플 언패킹
x, y = coordinates
print(f"x 좌표: {x}, y 좌표: {y}")

# 언패킹으로 변수 교환
a, b = 10, 20
a, b = b, a # (20, 10) 튜플이 생성되어 언패킹됨
print(f"교환 후: a={a}, b={b}")

15. 튜플의 활용

튜플은 불변성이라는 특징 덕분에 함수에서 여러 개의 값을 안전하게 반환하거나, 딕셔너리의 키로 사용되는 등 특별한 용도로 활용됩니다.

언제 튜플을 사용할까?

프로그램 실행 중에 절대 변하면 안 되는 데이터(예: 좌표, 설정값, 함수의 반환값)를 저장할 때 리스트 대신 튜플을 사용하면 데이터의 무결성을 보장할 수 있습니다.

# 9. 함수에서 여러 값 반환
def get_student_info():
    return "홍길동", 25, "컴퓨터공학과"

# 함수 결과를 언패킹으로 받기

함수는 특정 작업을 수행하는 코드 묶음입니다. `def` 키워드로 정의하며, 코드의 재사용성을 높이고 프로그램을 구조적으로 만들어줍니다.

함수의 구성 요소

17. 클래스 기본 구조

클래스는 객체를 만들기 위한 '설계도'입니다. 관련된 데이터(속성)와 기능(메서드)을 하나로 묶어 관리할 수 있습니다.

주요 용어

  • 객체(Object): 클래스라는 설계도로 만들어진 실체.
  • 속성(Attribute): 객체가 가지는 데이터. (예: `self.name`)
  • 메서드(Method): 객체가 수행할 수 있는 동작(함수).
  • `__init__`: 객체가 생성될 때 속성을 초기화하는 특별한 메서드.
class ClassName:
    # 초기화 메서드 (객체 생성 시 호출)
    def __init__(self, param1, param2):

        self.attribute1 = param1
        self.attribute2 = param2

    # 메서드 (객체의 기능)
    def method_name(self, param):
        print(f"메서드 호출됨: {self.attribute1}")
        
# 클래스로부터 객체(인스턴스)를 생성

my_object.method_name("전달값")

18. 클래스 활용 예제

클래스를 조합하여 더 복잡한 객체를 만들 수 있습니다. 예를 들어, `Point2D`라는 점 클래스를 부품으로 사용하여 `Figure`(도형) 클래스를 조립할 수 있습니다.

객체 지향 프로그래밍 (OOP)

이처럼 독립적인 객체들을 만들고, 이 객체들이 서로 상호작용하도록 프로그램을 설계하는 방식을 객체 지향 프로그래밍이라고 합니다.

class Point2D:
    def __init__(self, x, y):
        self.x, self.y = x, y

class Figure:
    def __init__(self, name):
        self.name = name
        self.points = {}

    def add_point(self, name, x, y):
        self.points[name] = Point2D(x, y)

    def show_info(self):
        print(f"--- 도형 '{self.name}' 정보 ---")
        for name, point in self.points.items():
            print(f"  - {name}: ({point.x}, {point.y})")

# 클래스 사용
person = Figure("사람1")
person.add_point("코", 95, 50)
person.add_point("왼쪽_어깨", 60, 80)
person.show_info()

20. 파일 읽고 쓰기

`open()` 함수를 사용하여 파일을 열고, `.read()`, `.write()` 등의 메서드로 내용을 읽거나 쓸 수 있습니다. `with`문을 사용하면 파일을 안전하게 자동으로 닫아줍니다.

파일 모드(mode)

  • `'r'`: 읽기 (Read) - 기본값
  • `'w'`: 쓰기 (Write) - 기존 내용을 덮어씀
  • `'a'`: 추가 (Append) - 파일 끝에 이어씀
  • `encoding='utf-8'`: 한글 깨짐 방지를 위해 필수
# 1. 텍스트 파일 쓰기 (w: write)
filename = "data/sample.txt"
content = "안녕하세요, 파이썬!"

with open(filename, "w", encoding="utf-8") as file:
    file.write(content)

# 2. 텍스트 파일 읽기 (r: read)
with open(filename, "r", encoding="utf-8") as file:
    read_content = file.read()
    print(f"파일 내용: {read_content}")

# 4. 파일에 내용 추가하기 (a: append)
additional = "\n이 줄은 추가되었습니다."
with open(filename, "a", encoding="utf-8") as file:
    file.write(additional)

21. 파일 입출력 활용

파일 입출력은 데이터를 영구적으로 저장하고 관리하는 핵심 기술입니다. CSV 형태의 데이터를 파일로 저장하고, 이를 다시 읽어와 통계를 내는 등 다양하게 활용할 수 있습니다.

한 줄씩 처리하기

대용량 파일을 다룰 때는 `.read()`로 전체를 읽는 대신 `for line in file:` 구문을 사용하여 한 줄씩 처리하는 것이 메모리 효율에 좋습니다.

# 5. 학생 정보 파일 만들기 (CSV 형태)

with open(student_file, "w", encoding="utf-8") as file:

22. 파일 시스템 탐색

`os` 모듈의 다양한 함수를 이용해 현재 디렉토리의 파일 목록을 얻거나, 파일의 크기, 수정 시간 등의 상세 정보를 확인할 수 있습니다.

자주 쓰이는 `os` 모듈 함수

  • `os.listdir(path)`: 경로의 파일/폴더 목록 반환
  • `os.path.isfile(path)`: 파일인지 확인
  • `os.path.isdir(path)`: 폴더인지 확인
  • `os.path.getsize(path)`: 파일 크기 반환
import os

# 1. 현재 디렉토리 탐색
current_path = "."
items = os.listdir(current_path)
print(f"현재 위치의 항목 수: {len(items)}")

# 2. 파일 상세 정보 확인
for item in items[:3]: # 3개만 예시로
    if os.path.isfile(item):
        size = os.path.getsize(item)
        print(f"📄 {item} ({size} bytes)")
    elif os.path.isdir(item):
        print(f"📁 {item}")

# 3. 특정 확장자 파일 찾기
py_files = [f for f in items if f.endswith('.py')]
print(f"\n.py 파일 개수: {len(py_files)}")
