실시간 객체 탐지 - YOLO 시작하기

1. YOLO (You Only Look Once)

YOLO는 이미지나 영상에서 객체의 위치와 종류를 빠르고 정확하게 찾아내는 딥러닝 기반의 객체 탐지 모델입니다. 특히 실시간 처리에 강점을 보여 웹캠, CCTV 등 다양한 분야에서 활용됩니다.

라이브러리 설치

YOLO 모델을 쉽게 사용할 수 있도록 패키징된 `ultralytics` 라이브러리를 `pip`를 통해 설치합니다. OpenCV도 함께 설치해야 합니다.

# ultralytics와 OpenCV 설치 명령어
# 터미널(cmd, PowerShell)에 아래와 같이 입력하세요.

pip install ultralytics
pip install opencv-python

2. 기본 이미지 객체 탐지

로드된 YOLO 모델에 이미지를 입력하면, 이미지 내에 존재하는 객체들의 정보(클래스, 위치, 신뢰도)를 담은 결과값을 반환합니다.

기본 탐지 과정

  1. `YOLO('yolov8n.pt')`로 모델을 로드합니다.
  2. `cv2.imread()`로 이미지를 불러옵니다.
  3. `model(image)`를 호출하여 탐지를 실행합니다.
  4. 결과 객체의 `.plot()` 메서드로 탐지 결과를 시각화합니다.
  5. `cv2.imshow()`로 결과를 화면에 표시합니다.
import cv2
from ultralytics import YOLO

# 1. 모델 로드
model = YOLO('yolov8n.pt')

# 2. 이미지 로드
image = cv2.imread('pointing_up.jpg')

# 3. 객체 탐지 실행
results = model(image)

# 4. 결과 이미지 생성 및 표시
annotated_frame = results[0].plot()
cv2.imshow('YOLO Detection', annotated_frame)

cv2.waitKey(0)
cv2.destroyAllWindows()

3. 실시간 웹캠 객체 탐지

OpenCV를 사용하여 웹캠으로부터 실시간으로 영상 프레임을 받아오고, 각 프레임에 대해 YOLO 탐지를 수행하여 실시간 객체 탐지를 구현할 수 있습니다.

실시간 처리의 핵심

`while` 루프 안에서 `cap.read()`로 프레임을 계속 읽어오고, `model(frame)`으로 매 프레임마다 탐지를 실행하는 것이 핵심 구조입니다. `cv2.waitKey(1)`은 프레임 간의 짧은 지연을 주어 영상이 보이게 합니다.

import cv2
from ultralytics import YOLO

model = YOLO('yolov8n.pt')
cap = cv2.VideoCapture(0) # 0번 웹캠

while cap.isOpened():
    ret, frame = cap.read()
    if not ret:
        break

    # 실시간으로 객체 탐지
    results = model(frame, verbose=False)
    annotated_frame = results[0].plot()
    
    cv2.imshow('YOLO Real-time', annotated_frame)

    # 'q' 키를 누르면 종료
    if cv2.waitKey(1) & 0xFF == ord('q'):
        break

cap.release()
cv2.destroyAllWindows()

4. 특정 객체 필터링 및 카운팅

탐지 결과(`results`) 객체에서 클래스 ID(`box.cls`)와 신뢰도(`box.conf`) 정보를 추출하여, 우리가 원하는 특정 객체만 필터링하고 그 개수를 셀 수 있습니다.

결과 데이터 접근

`results[0].boxes`를 통해 탐지된 모든 객체의 바운딩 박스 정보에 접근할 수 있습니다. 각 `box` 객체는 위치(`xyxy`), 신뢰도(`conf`), 클래스 ID(`cls`) 등의 속성을 가집니다.

# ... (모델 로드 및 웹캠 연결) ...

TARGET_CLASSES = {0: 'person', 2: 'car'}

while True:
    ret, frame = cap.read()
    if not ret: break

    results = model(frame, verbose=False)
    person_count = 0
    
    if results[0].boxes is not None:
        for box in results[0].boxes:
            class_id = int(box.cls)
            # 'person' 클래스(ID: 0)만 카운트
            if class_id == 0 and box.conf > 0.5:
                person_count += 1

    annotated_frame = results[0].plot()
    cv2.putText(annotated_frame, f'Person Count: {person_count}', 
               (10, 30), cv2.FONT_HERSHEY_SIMPLEX, 1, (0, 255, 0), 2)
    
    cv2.imshow('YOLO Counting', annotated_frame)
    if cv2.waitKey(1) & 0xFF == ord('q'): break

5. 응용: 스마트 보안 시스템

지금까지 배운 기술들을 클래스로 묶어 체계적인 애플리케이션을 만들 수 있습니다. 특정 객체(예: 사람)가 탐지되면 알림을 보내고, 해당 장면을 스크린샷으로 자동 저장하는 스마트 보안 시스템을 구현할 수 있습니다.

시스템 설계 아이디어

  • 관심 객체 정의: 'person', 'car' 등 보안상 중요한 객체 목록을 정의합니다.
  • 알림 조건 설정: 특정 객체가 일정 신뢰도 이상으로 탐지될 때를 이벤트로 간주합니다.
  • 쿨다운 적용: 중복 알림을 방지하기 위해 한 번 알림 후 일정 시간 대기합니다.
  • 증거 저장: 이벤트 발생 시 `cv2.imwrite()`로 해당 프레임을 저장합니다.
