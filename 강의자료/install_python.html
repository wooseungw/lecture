개발 환경 설정 - Python 설치

1단계: 공식 설치 파일 다운로드

가장 먼저, Python 공식 홈페이지에 접속하여 설치 파일을 다운로드해야 합니다. 브라우저를 열고 python.org 로 이동하세요.

메인 화면의 'Downloads' 메뉴에서 최신 버전의 설치 파일을 클릭하여 다운로드합니다.

공식 홈페이지를 이용하는 이유

공식 홈페이지는 바이러스나 악성 코드로부터 가장 안전하며, 안정적인 최신 버전을 제공합니다. 비공식 경로의 파일은 보안에 취약할 수 있으니 항상 공식 사이트를 이용하세요.

python.org 홈페이지의 다운로드 버튼 강조 이미지

2단계: 설치 옵션 선택 (매우 중요)

다운로드한 설치 파일을 실행하면 설정 창이 나타납니다. 여기서 가장 중요한 단계는 설치 프로그램 창 하단에 있는 'Add python.exe to PATH' 옵션을 체크하는 것입니다.

이 옵션을 반드시 체크한 후, 'Install Now'를 클릭하여 다음 단계로 진행하세요.

'PATH'에 추가란?

운영체제가 'python' 명령어 실행 파일을 찾을 수 있도록 경로를 등록하는 과정입니다. 이 설정을 해야 터미널의 어느 위치에서든 python을 바로 실행할 수 있어 편리합니다.

'Add python.exe to PATH' 체크박스를 강조한 이미지

개발 환경 설정 - Python 설치

3단계: 설치 진행 및 완료

'Install Now'를 클릭하면 설치가 자동으로 진행됩니다. 컴퓨터 사양에 따라 몇 분 정도 소요될 수 있습니다.

설치가 끝나면 'Setup was successful' 이라는 메시지가 나타납니다. 'Close' 버튼을 눌러 설치 프로그램을 닫습니다.

경로 길이 제한 해제 (선택 사항)

완료 화면에 'Disable path length limit' 버튼이 보일 수 있습니다. 이는 260자 이상의 긴 파일 경로를 사용할 수 있게 하는 설정으로, 클릭해두면 나중에 발생할 수 있는 문제를 예방할 수 있습니다.

개발 환경 설정 - Python 설치

4단계: 터미널에서 설치 확인

이제 Python이 잘 설치되었는지 최종 확인할 차례입니다. 시작 메뉴에서 'cmd' 또는 'PowerShell'을 검색하여 터미널을 실행하세요.

터미널 창에 python --version 을 입력하고 Enter를 누르면 설치된 버전이 표시됩니다. 이어서 pip --version 도 확인합니다.

무엇을 확인한 건가요?

python은 프로그래밍 언어 자체를, pip는 다른 개발자들이 만든 유용한 도구(패키지)를 설치하는 관리자입니다. 둘 다 정상적으로 확인되면 모든 준비가 끝난 것입니다.
