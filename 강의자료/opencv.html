1. OpenCV 라이브러리

OpenCV는 실시간 컴퓨터 비전(Computer Vision)에 중점을 둔 프로그래밍 라이브러리입니다. 이미지 처리뿐만 아니라 동영상, 웹캠 등 비디오 스트림을 다루는 데 강력한 기능을 제공합니다.

라이브러리 설치

OpenCV는 `pip`를 사용하여 설치할 수 있습니다. Pillow와 달리 NumPy 라이브러리에 의존하므로 함께 설치되는 경우가 많습니다.

# OpenCV 라이브러리 설치 명령어
# 터미널(cmd, PowerShell)에 아래와 같이 입력하세요.

pip install opencv-python

# ---------------------------------

# cv2 라는 별칭으로 import하여 사용합니다.

print(f"OpenCV 버전: {cv2.__version__}")

2. 이미지 읽고 화면에 표시하기

`cv2.imread()` 함수로 이미지를 읽어 NumPy 배열 형태로 가져옵니다. `cv2.imshow()` 함수는 이 배열을 해석하여 새 창에 이미지로 표시합니다.

`waitKey`와 `destroyAllWindows`

  • cv2.waitKey(0): 사용자가 키보드를 누를 때까지 무한정 대기합니다. 이 코드가 없으면 창이 바로 닫혀 이미지를 볼 수 없습니다.
  • cv2.destroyAllWindows(): 눌린 키에 상관없이 열려있는 모든 OpenCV 창을 닫습니다.
import cv2

# 이미지 파일 경로
image_path = 'ex_img/thumbs_up.jpg'

# 1. 이미지 읽기 (NumPy 배열로 반환)
image = cv2.imread(image_path)

# 2. 'Image'라는 이름의 창에 이미지 표시
cv2.imshow('Image', image)

# 3. 키 입력 대기
cv2.waitKey(0)

# 4. 모든 창 닫기
cv2.destroyAllWindows()

3. 이미지 저장 및 색상 공간 변환

`cv2.imwrite()` 함수로 메모리에 있는 이미지(NumPy 배열)를 파일로 저장할 수 있습니다. `cv2.cvtColor()` 함수는 이미지의 색상 공간을 변환하는 데 사용됩니다.

BGR vs RGB

OpenCV는 전통적으로 색상 채널을 BGR(파랑, 초록, 빨강) 순서로 처리합니다. Pillow나 Matplotlib 같은 다른 라이브러리는 RGB 순서를 사용하므로, 함께 사용할 때 색상 왜곡이 발생하지 않도록 주의해야 합니다.

import cv2

image = cv2.imread('ex_img/thumbs_up.jpg')

# 1. 이미지 색상 변환 (BGR -> GRAY)
gray_image = cv2.cvtColor(image, cv2.COLOR_BGR2GRAY)

# 2. 변환된 이미지 저장
output_path = 'ex_img/thumbs_up_gray.jpg'
cv2.imwrite(output_path, gray_image)

print(f"그레이스케일 이미지 저장 완료: {output_path}")

# 변환된 이미지 화면에 표시
cv2.imshow('Grayscale Image', gray_image)
cv2.waitKey(0)
cv2.destroyAllWindows()

4. 동영상 파일 재생하기

`cv2.VideoCapture('파일경로')`로 비디오 파일을 엽니다. `while` 반복문 안에서 `.read()` 메서드를 호출하여 비디오의 각 프레임을 연속적으로 읽어와 화면에 표시할 수 있습니다.

`waitKey`의 역할

`cv2.waitKey(25)`는 25밀리초 동안 키 입력을 기다립니다. 이 지연 시간 동안 프레임이 화면에 표시되며, 이 시간이 비디오의 재생 속도를 결정합니다. `q` 키를 누르면 반복문을 탈출합니다.

import cv2

# 비디오 파일 열기
cap = cv2.VideoCapture('ex_img/sample_video.mp4')

while cap.isOpened():
    # 프레임 단위로 읽기
    ret, frame = cap.read()
    if not ret: # 더 이상 프레임이 없으면 종료
        break

    cv2.imshow('Video', frame)

    # 'q' 키를 누르면 종료
    if cv2.waitKey(25) & 0xFF == ord('q'):
        break

# 자원 해제
cap.release()
cv2.destroyAllWindows()

이미지/영상 처리 - 비디오 처리

5. 웹캠 실시간 처리하기

`cv2.VideoCapture(0)`와 같이 파일 경로 대신 카메라 번호(보통 0)를 전달하면 웹캠을 열 수 있습니다. 동영상 파일 처리와 동일한 방식으로 실시간 영상 스트림을 다룰 수 있습니다.

실시간 처리의 응용

웹캠으로 들어오는 각 프레임에 대해 얼굴 인식, 객체 추적, 필터 적용 등 다양한 컴퓨터 비전 기술을 실시간으로 적용할 수 있습니다.

import cv2

# 0번 웹캠 열기
cap = cv2.VideoCapture(0)

while cap.isOpened():
    ret, frame = cap.read()
    if not ret:
        break
    
    # (선택) 프레임 좌우 반전
    frame = cv2.flip(frame, 1)

    cv2.imshow('Webcam', frame)

    if cv2.waitKey(1) & 0xFF == ord('q'):
        break

cap.release()
cv2.destroyAllWindows()

6. 도형 그리기

OpenCV는 이미지 위에 사각형, 원, 선 등 다양한 도형을 그리는 기능을 제공합니다. 이를 통해 이미지에 정보를 시각적으로 추가하거나 마스킹 등의 작업을 수행할 수 있습니다.

도형 그리기 함수

  • cv2.rectangle(img, pt1, pt2, color, thickness): 사각형 그리기
  • cv2.circle(img, center, radius, color, thickness): 원 그리기
  • cv2.line(img, pt1, pt2, color, thickness): 선 그리기
  • thickness: 선의 두께. -1을 지정하면 도형 내부를 채웁니다.
import cv2
import numpy as np

# 검은색 배경 이미지 생성 (500x500, 3채널)
img = np.zeros((500, 500, 3), dtype=np.uint8)

# 1. 사각형 그리기
cv2.rectangle(img, (50, 50), (200, 150), (0, 255, 0), 2)  # 초록색 테두리
cv2.rectangle(img, (250, 50), (400, 150), (255, 0, 0), -1)  # 파란색 채우기

# 2. 원 그리기
cv2.circle(img, (125, 250), 50, (0, 255, 255), 3)  # 노란색 테두리
cv2.circle(img, (325, 250), 50, (255, 255, 0), -1)  # 청록색 채우기

# 3. 선 그리기
cv2.line(img, (50, 350), (200, 350), (255, 255, 255), 2)  # 가로선 (흰색)
cv2.line(img, (250, 330), (400, 370), (255, 0, 255), 3)  # 대각선 (자홍색)

# 이미지 출력
cv2.imshow('Shape Drawing Examples', img)
cv2.waitKey(0)
cv2.destroyAllWindows()

7. 텍스트 그리기

OpenCV의 `cv2.putText()` 함수를 사용하여 이미지에 텍스트를 추가할 수 있습니다. 다양한 폰트, 크기, 색상, 두께를 설정하여 텍스트를 커스터마이징할 수 있습니다.

`cv2.putText()` 파라미터

  • image: 텍스트를 그릴 이미지
  • text: 그릴 텍스트 (문자열)
  • position: 텍스트 시작 위치 (x, y)
  • font: 폰트 타입 (예: `cv2.FONT_HERSHEY_SIMPLEX`)
  • scale: 폰트 크기 배율
  • color: 텍스트 색상 (B, G, R)
  • thickness: 텍스트 두께
  • lineType: 선의 타입 (예: `cv2.LINE_AA`는 안티앨리어싱 적용)
import cv2
import numpy as np

# 흰색 배경 이미지 생성
img = np.ones((400, 600, 3), dtype=np.uint8) * 255

# 텍스트 그리기
cv2.putText(img, 'Hello, OpenCV!', (50, 100), 
           cv2.FONT_HERSHEY_SIMPLEX, 1.5, (255, 0, 0), 2, cv2.LINE_AA)

cv2.putText(img, 'Python & Computer Vision', (50, 200), 
           cv2.FONT_HERSHEY_COMPLEX, 1, (0, 128, 0), 3)

cv2.putText(img, 'Gemini CLI', (50, 300), 
           cv2.FONT_HERSHEY_SCRIPT_COMPLEX, 2, (0, 0, 255), 4)

# 이미지 출력
cv2.imshow('Text Examples', img)
cv2.waitKey(0)
cv2.destroyAllWindows()

이미지/영상 처리 - 이미지 변환

8. 이미지 크기 조절 및 블러

이미지의 크기를 조절하거나(리사이즈), 블러(흐림) 효과를 적용하여 노이즈를 줄이거나 특정 부분을 강조할 수 있습니다. 다양한 보간법과 블러 필터가 제공됩니다.

주요 변환 함수

  • cv2.resize(src, dsize, fx, fy, interpolation): 이미지 크기 조절
  • cv2.GaussianBlur(src, ksize, sigmaX): 가우시안 블러
  • cv2.blur(src, ksize): 평균 블러
  • cv2.medianBlur(src, ksize): 미디언 블러 (노이즈 제거에 효과적)
import cv2
import numpy as np

# 원본 이미지 로드 (또는 생성)
original_image = cv2.imread('ex_img/thumbs_up.jpg')
if original_image is None:
    original_image = np.zeros((300, 400, 3), dtype=np.uint8)
    cv2.putText(original_image, 'No Image', (50, 150), 
               cv2.FONT_HERSHEY_SIMPLEX, 1, (255, 255, 255), 2)

# 1. 이미지 크기 조절
resized_image = cv2.resize(original_image, (200, 200))

# 2. 가우시안 블러 적용
blurred_image = cv2.GaussianBlur(original_image, (15, 15), 0)

# 결과 이미지들을 하나의 창에 표시
combined_image = np.hstack((original_image, resized_image, blurred_image))

cv2.imshow('Image Transformations', combined_image)
cv2.waitKey(0)
cv2.destroyAllWindows()

이미지/영상 처리 - 필터 및 효과

9. 다양한 이미지 필터 및 효과

OpenCV는 에지 검출, 형태학적 연산, 색상 공간 변환 등 다양한 이미지 필터와 효과를 제공하여 이미지의 특징을 추출하거나 개선할 수 있습니다.

주요 필터 및 효과

  • cv2.Canny(): Canny 에지 검출 (이미지의 윤곽선 찾기)
  • cv2.erode() / cv2.dilate(): 침식/팽창 (객체 크기 조절, 노이즈 제거)
  • cv2.cvtColor(img, cv2.COLOR_BGR2HSV): 색상 공간 변환 (HSV는 색조, 채도, 명도 분리)
  • 히스토그램 평활화: 이미지의 대비를 개선
import cv2
import numpy as np

# 테스트 이미지 생성 (또는 로드)
test_image = cv2.imread('ex_img/thumbs_up.jpg')
if test_image is None:
    test_image = np.zeros((300, 400, 3), dtype=np.uint8)
    cv2.putText(test_image, 'No Image', (50, 150), 
               cv2.FONT_HERSHEY_SIMPLEX, 1, (255, 255, 255), 2)

# 1. Canny 에지 검출
gray_image = cv2.cvtColor(test_image, cv2.COLOR_BGR2GRAY)
canny_edges = cv2.Canny(gray_image, 100, 200)

# 2. 채도 증가 (HSV 변환)
hsv_image = cv2.cvtColor(test_image, cv2.COLOR_BGR2HSV)
hsv_image[:, :, 1] = np.clip(hsv_image[:, :, 1] * 1.5, 0, 255)
saturated_image = cv2.cvtColor(hsv_image, cv2.COLOR_HSV2BGR)

# 결과 이미지들을 하나의 창에 표시
combined_image = np.hstack((test_image, 
                            cv2.cvtColor(canny_edges, cv2.COLOR_GRAY2BGR), 
                            saturated_image))

cv2.imshow('Image Filters & Effects', combined_image)
cv2.waitKey(0)
cv2.destroyAllWindows()

10. 마우스 이벤트 처리

OpenCV는 마우스 이벤트를 감지하고 처리하는 기능을 제공하여 사용자와의 상호작용이 필요한 애플리케이션을 만들 수 있습니다. 이를 통해 그림판과 같은 기능을 구현할 수 있습니다.

`cv2.setMouseCallback()`

  • cv2.setMouseCallback(window_name, callback_function): 특정 창에 마우스 이벤트 콜백 함수를 등록합니다.
  • 콜백 함수는 이벤트 타입, 마우스 좌표(x, y), 플래그(키보드 상태), 사용자 정의 파라미터 등을 인자로 받습니다.
import cv2
import numpy as np

# 전역 변수들
drawing = False  # 마우스 드래그 중인지 확인
ix, iy = -1, -1  # 시작 좌표

# 이미지 생성
img = np.ones((400, 600, 3), dtype=np.uint8) * 255

# 마우스 콜백 함수
def mouse_callback(event, x, y, flags, param):
    global ix, iy, drawing, img
    
    if event == cv2.EVENT_LBUTTONDOWN:
        drawing = True
        ix, iy = x, y
    elif event == cv2.EVENT_MOUSEMOVE:
        if drawing:
            cv2.rectangle(img, (ix, iy), (x, y), (0, 255, 0), -1)
    elif event == cv2.EVENT_LBUTTONUP:
        drawing = False
        cv2.rectangle(img, (ix, iy), (x, y), (0, 255, 0), -1)

# 창 생성 및 마우스 콜백 설정
cv2.namedWindow('Drawing Canvas')
cv2.setMouseCallback('Drawing Canvas', mouse_callback)

while True:
    cv2.imshow('Drawing Canvas', img)
    if cv2.waitKey(1) & 0xFF == ord('q'):
        break

cv2.destroyAllWindows()

이미지/영상 처리 - 고급 필터링

11. 고급 필터링 및 효과

OpenCV는 이미지의 픽셀 값을 직접 조작하거나, 복잡한 수학적 연산을 통해 다양한 시각적 효과를 구현할 수 있는 고급 필터링 기법을 제공합니다.

주요 고급 필터

  • cv2.filter2D(): 사용자 정의 커널을 이용한 2D 컨볼루션 필터
  • cv2.warpAffine() / cv2.warpPerspective(): 아핀/원근 변환 (이미지 왜곡, 회전 등)
  • 색상 팔레트 적용: 그레이스케일 이미지에 다양한 색상 맵을 적용하여 시각화
import cv2
import numpy as np

# 원본 이미지 로드 (또는 생성)
original_image = cv2.imread('ex_img/thumbs_up.jpg')
if original_image is None:
    original_image = np.zeros((300, 400, 3), dtype=np.uint8)
    cv2.putText(original_image, 'No Image', (50, 150), 
               cv2.FONT_HERSHEY_SIMPLEX, 1, (255, 255, 255), 2)

# 1. 샤프닝 필터
kernel_sharpening = np.array([[-1, -1, -1],
                              [-1, 9, -1],
                              [-1, -1, -1]])
sharpened_image = cv2.filter2D(original_image, -1, kernel_sharpening)

# 2. 엠보싱 필터
kernel_emboss = np.array([[-2, -1, 0],
                          [-1, 1, 1],
                          [0, 1, 2]])
embossed_image = cv2.filter2D(original_image, -1, kernel_emboss)
embossed_image = cv2.cvtColor(embossed_image, cv2.COLOR_BGR2GRAY)
embossed_image = cv2.cvtColor(embossed_image, cv2.COLOR_GRAY2BGR) # 3채널로 변환

# 3. 색상 팔레트 적용 (그레이스케일 이미지에)
gray_original = cv2.cvtColor(original_image, cv2.COLOR_BGR2GRAY)
colored_map_image = cv2.applyColorMap(gray_original, cv2.COLORMAP_JET)

# 결과 이미지들을 하나의 창에 표시
combined_image = np.hstack((original_image, sharpened_image, embossed_image, colored_map_image))

cv2.imshow('Advanced Filters & Effects', combined_image)
cv2.waitKey(0)
cv2.destroyAllWindows()
