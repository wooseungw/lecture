1. OpenCV 라이브러리

OpenCV는 실시간 컴퓨터 비전(Computer Vision)에 중점을 둔 프로그래밍 라이브러리입니다. 이미지 처리뿐만 아니라 동영상, 웹캠 등 비디오 스트림을 다루는 데 강력한 기능을 제공합니다.

라이브러리 설치

OpenCV는 `pip`를 사용하여 설치할 수 있습니다. Pillow와 달리 NumPy 라이브러리에 의존하므로 함께 설치되는 경우가 많습니다.

# OpenCV 라이브러리 설치 명령어
# 터미널(cmd, PowerShell)에 아래와 같이 입력하세요.

pip install opencv-python

2. 이미지 읽고 화면에 표시하기

`cv2.imread()` 함수로 이미지를 읽어 NumPy 배열 형태로 가져옵니다. `cv2.imshow()` 함수는 이 배열을 해석하여 새 창에 이미지로 표시합니다.

`waitKey`와 `destroyAllWindows`

  • cv2.waitKey(0): 사용자가 키보드를 누를 때까지 무한정 대기합니다. 이 코드가 없으면 창이 바로 닫혀 이미지를 볼 수 없습니다.
  • cv2.destroyAllWindows(): 눌린 키에 상관없이 열려있는 모든 OpenCV 창을 닫습니다.
import cv2

# 이미지 파일 경로
image_path = 'ex_img/thumbs_up.jpg'

# 1. 이미지 읽기 (NumPy 배열로 반환)
image = cv2.imread(image_path)

# 2. 'Image'라는 이름의 창에 이미지 표시
cv2.imshow('Image', image)

# 3. 키 입력 대기
cv2.waitKey(0)

# 4. 모든 창 닫기
cv2.destroyAllWindows()

3. 이미지 저장 및 색상 공간 변환

`cv2.imwrite()` 함수로 메모리에 있는 이미지(NumPy 배열)를 파일로 저장할 수 있습니다. `cv2.cvtColor()` 함수는 이미지의 색상 공간을 변환하는 데 사용됩니다.

BGR vs RGB

OpenCV는 전통적으로 색상 채널을 BGR(파랑, 초록, 빨강) 순서로 처리합니다. Pillow나 Matplotlib 같은 다른 라이브러리는 RGB 순서를 사용하므로, 함께 사용할 때 색상 왜곡이 발생하지 않도록 주의해야 합니다.

import cv2

image = cv2.imread('ex_img/thumbs_up.jpg')

# 1. 이미지 색상 변환 (BGR -> GRAY)
gray_image = cv2.cvtColor(image, cv2.COLOR_BGR2GRAY)

# 2. 변환된 이미지 저장
output_path = 'ex_img/thumbs_up_gray.jpg'
cv2.imwrite(output_path, gray_image)

print(f"그레이스케일 이미지 저장 완료: {output_path}")

# 변환된 이미지 화면에 표시
cv2.imshow('Grayscale Image', gray_image)
cv2.waitKey(0)
cv2.destroyAllWindows()

4. 동영상 파일 재생하기

`cv2.VideoCapture('파일경로')`로 비디오 파일을 엽니다. `while` 반복문 안에서 `.read()` 메서드를 호출하여 비디오의 각 프레임을 연속적으로 읽어와 화면에 표시할 수 있습니다.

`waitKey`의 역할

`cv2.waitKey(25)`는 25밀리초 동안 키 입력을 기다립니다. 이 지연 시간 동안 프레임이 화면에 표시되며, 이 시간이 비디오의 재생 속도를 결정합니다. `q` 키를 누르면 반복문을 탈출합니다.

import cv2

# 비디오 파일 열기
cap = cv2.VideoCapture('ex_img/sample_video.mp4')

while cap.isOpened():
    # 프레임 단위로 읽기
    ret, frame = cap.read()
    if not ret: # 더 이상 프레임이 없으면 종료
        break

    cv2.imshow('Video', frame)

    # 'q' 키를 누르면 종료
    if cv2.waitKey(25) & 0xFF == ord('q'):
        break

# 자원 해제
cap.release()
cv2.destroyAllWindows()

이미지/영상 처리 - 비디오 처리

5. 웹캠 실시간 처리하기

`cv2.VideoCapture(0)`와 같이 파일 경로 대신 카메라 번호(보통 0)를 전달하면 웹캠을 열 수 있습니다. 동영상 파일 처리와 동일한 방식으로 실시간 영상 스트림을 다룰 수 있습니다.

실시간 처리의 응용

웹캠으로 들어오는 각 프레임에 대해 얼굴 인식, 객체 추적, 필터 적용 등 다양한 컴퓨터 비전 기술을 실시간으로 적용할 수 있습니다.

import cv2

# 0번 웹캠 열기
cap = cv2.VideoCapture(0)

while cap.isOpened():
    ret, frame = cap.read()
    if not ret:
        break
    
    # (선택) 프레임 좌우 반전
    frame = cv2.flip(frame, 1)

    cv2.imshow('Webcam', frame)

    if cv2.waitKey(1) & 0xFF == ord('q'):
        break

cap.release()
cv2.destroyAllWindows()
