1. OpenCV 라이브러리란?

OpenCV(Open Source Computer Vision Library)는 실시간 컴퓨터 비전에 중점을 둔 프로그래밍 라이브러리입니다. 이미지 처리, 동영상 처리, 웹캠 실시간 처리 등 다양한 컴퓨터 비전 작업을 쉽게 구현할 수 있습니다.

처음엔 터미널에서 pip 설치만 확인하고, 간단한 import와 버전 출력으로 환경이 정상인지 빠르게 점검하세요.

좌표계는 좌상단이 (0,0)이며 색상은 (B,G,R) 순서입니다. thickness=-1로 채우기, putText로 제목을 추가해보세요.

Canny는 그레이스케일+가우시안 블러 전처리 후 임계값을 1:2~1:3 비율로 조절하며 결과를 비교해 보세요.

📥 라이브러리 설치

OpenCV는 `pip`를 사용하여 간단히 설치할 수 있습니다. NumPy 라이브러리에 의존하므로 함께 설치되며, 이미지를 NumPy 배열로 처리합니다.

2. 이미지 읽고 화면에 표시하기

OpenCV의 가장 기본적인 기능입니다. `cv2.imread()`로 이미지를 NumPy 배열로 읽어오고, `cv2.imshow()`로 화면에 표시합니다. 이것이 모든 이미지 처리의 시작점입니다.

imshow 후에는 반드시 waitKey와 destroyAllWindows를 호출해야 창이 보입니다. 이미지 경로가 틀리면 imread가 None을 반환합니다.

⌨️ 필수 함수들

  • cv2.waitKey(0): 키보드 입력을 대기합니다. 0은 무한 대기를 의미합니다.
  • cv2.destroyAllWindows(): 모든 OpenCV 창을 닫습니다.
  • 이 두 함수가 없으면 창이 바로 사라져서 이미지를 볼 수 없습니다!
import cv2

# 이미지 파일 경로 설정
image_path = 'ex_img/thumbs_up.jpg'

# 1. 이미지 읽기 (NumPy 배열로 반환)
image = cv2.imread(image_path)

# 이미지 로드 확인
if image is None:
    print("이미지를 찾을 수 없습니다!")
else:
    # 2. 'Image'라는 제목의 창에 이미지 표시
    cv2.imshow('Image', image)

    # 3. 키 입력을 기다림
    cv2.waitKey(0)

    # 4. 모든 OpenCV 창 닫기
    cv2.destroyAllWindows()

이미지/영상 처리 - 이미지 변환

3. 이미지 저장 및 색상 공간 변환

이미지를 다른 색상 공간으로 변환하고 파일로 저장하는 방법입니다. 그레이스케일 변환이 가장 기본적이며, 많은 이미지 처리 작업의 전처리 단계입니다.

OpenCV의 색상 순서는 BGR입니다. 그레이스케일은 용량을 줄이고 에지/임계값 처리에 유리하니 자주 활용해보세요.

🎨 색상 변환

OpenCV는 BGR 순서로 색상을 저장합니다. `cv2.COLOR_BGR2GRAY`는 컬러 이미지를 흑백으로 변환하는 가장 일반적인 변환입니다.

import cv2

# 원본 이미지 읽기
image = cv2.imread('ex_img/thumbs_up.jpg')

# 1. 색상 변환 (BGR → 그레이스케일)
gray_image = cv2.cvtColor(image, cv2.COLOR_BGR2GRAY)

# 2. 변환된 이미지를 파일로 저장
output_path = 'ex_img/thumbs_up_gray.jpg'
cv2.imwrite(output_path, gray_image)

print("그레이스케일 이미지 저장 완료!")

# 원본과 변환된 이미지 동시 표시
cv2.imshow('Original', image)
cv2.imshow('Grayscale', gray_image)

cv2.waitKey(0)
cv2.destroyAllWindows()

이미지/영상 처리 - 비디오 처리

4. 동영상 파일 재생하기

비디오는 연속된 이미지(프레임)의 집합입니다. `VideoCapture`로 비디오를 열고, `while` 루프에서 프레임을 하나씩 읽어와 표시하면 동영상이 재생됩니다.

VideoCapture가 안 열리면 경로/코덱을 확인하고, 루프에서 ret이 False면 파일 끝 또는 오류입니다. 'q'키로 중단을 구현하세요.

⏱️ 재생 속도 조절

`waitKey(25)`는 25밀리초 대기하므로 약 40fps로 재생됩니다. 숫자를 줄이면 빨라지고, 늘리면 느려집니다. 'q' 키로 재생을 중단할 수 있습니다.

import cv2

# 비디오 파일 열기
cap = cv2.VideoCapture('ex_img/sample_video.mp4')

# 비디오 파일이 제대로 열렸는지 확인
if not cap.isOpened():
    print("비디오 파일을 열 수 없습니다!")
else:
    # 프레임 단위로 비디오 재생
    while cap.isOpened():
        ret, frame = cap.read()
        
        # 더 이상 읽을 프레임이 없으면 종료
        if not ret:
            print("비디오 재생 완료")
            break

        # 현재 프레임 표시
        cv2.imshow('Video', frame)

        # 'q' 키를 누르면 재생 중단
        if cv2.waitKey(25) & 0xFF == ord('q'):
            break

# 자원 해제
cap.release()
cv2.destroyAllWindows()

이미지/영상 처리 - 실시간 처리

5. 웹캠 실시간 처리하기

웹캠은 실시간 비디오 스트림을 제공합니다. VideoCapture(0)에서 0은 기본 웹캠을 의미합니다. 비디오 파일 처리와 동일한 방법으로 실시간 영상을 처리할 수 있습니다.

웹캠은 장치 번호(보통 0)로 엽니다. OS에선 카메라 권한을 허용해야 하며, 프레임 읽기 실패 시 즉시 루프를 빠져나오세요.

📹 실시간 처리 활용

웹캠의 각 프레임에 필터를 적용하거나, 얼굴 인식, 모션 디텍션 등 다양한 실시간 컴퓨터 비전 기법을 적용할 수 있습니다.

import cv2

# 0번 웹캠 열기 (기본 카메라)
cap = cv2.VideoCapture(0)

# 웹캠이 제대로 열렸는지 확인
if not cap.isOpened():
    print("웹캠을 열 수 없습니다!")
else:
    print("웹캠 시작됨. 'q' 키를 눌러 종료하세요.")
    
    while cap.isOpened():
        ret, frame = cap.read()
        
        if not ret:
            print("프레임을 읽을 수 없습니다!")
            break

        # 실시간 프레임 표시
        cv2.imshow('Webcam', frame)

        # 'q' 키로 종료 (1ms 대기)
        if cv2.waitKey(1) & 0xFF == ord('q'):
            break

# 웹캠 자원 해제
cap.release()
cv2.destroyAllWindows()

6. 다양한 도형 그리기

OpenCV는 이미지 위에 사각형, 원, 선, 다각형 등을 그릴 수 있습니다. 객체 검출 결과를 표시하거나 이미지에 정보를 시각적으로 추가할 때 매우 유용합니다.

  • cv2.rectangle(img, pt1, pt2, color, thickness): 사각형 그리기
  • cv2.circle(img, center, radius, color, thickness): 원 그리기
  • cv2.line(img, pt1, pt2, color, thickness): 선 그리기
  • thickness = -1이면 도형 내부를 색상으로 채웁니다

    • 🎨 도형 그리기 함수들

    • 모든 좌표는 좌상단 
    import cv2
import numpy as np

# 검은 배경 이미지 생성 (500x500, 3채널 BGR)
img = np.zeros((500, 500, 3), dtype=np.uint8)

print("=== OpenCV 도형 그리기 예제 ===")

# 1. 사각형 그리기
cv2.rectangle(img, (50, 50), (200, 150), (0, 255, 0), 2)   # 초록 테두리
cv2.rectangle(img, (250, 50), (400, 150), (255, 0, 0), -1) # 파랑 채우기

# 2. 원 그리기
cv2.circle(img, (125, 250), 50, (0, 255, 255), 3)  # 노란 테두리
cv2.circle(img, (325, 250), 50, (255, 255, 0), -1) # 청록 채우기

# 3. 선 그리기
cv2.line(img, (50, 350), (200, 350), (255, 255, 255), 2)  # 흰 가로선
cv2.line(img, (250, 330), (400, 370), (255, 0, 255), 3)   # 자홍 대각선

# 4. 다각형 그리기
points = np.array([[300, 400], [350, 450], [250, 450]], np.int32)
cv2.polylines(img, [points], True, (128, 255, 0), 2)

# 제목 추가
cv2.putText(img, 'OpenCV Shape Examples', (100, 30), 
           cv2.FONT_HERSHEY_SIMPLEX, 1, (255, 255, 255), 2)

cv2.imshow('Shape Drawing', img)
cv2.waitKey(0)
cv2.destroyAllWindows()

    7. 이미지에 텍스트 추가하기

    `cv2.putText()` 함수로 이미지에 다양한 텍스트를 그릴 수 있습니다. 객체 라벨링, 정보 표시, 디버깅 정보 출력 등에 광범위하게 사용됩니다.

    폰트/크기/색/두께 조합으로 가독성을 맞추세요. 작은 글자는 Line_AA로 가장자리를 부드럽게 하면 더 선명합니다.

    ✍️ 텍스트 옵션

    • 폰트: SIMPLEX, COMPLEX, SCRIPT_SIMPLEX 등
    • 크기: scale 매개변수로 조절
    • 색상: BGR 순서로 지정
    • 두께: thickness 매개변수로 조절
    import cv2
import numpy as np

# 흰색 배경 이미지 생성
img = np.ones((600, 800, 3), dtype=np.uint8) * 255

print("=== OpenCV 텍스트 그리기 예제 ===")

# 제목 텍스트
cv2.putText(img, 'OpenCV Text Examples', (200, 50), 
           cv2.FONT_HERSHEY_SIMPLEX, 1.5, (0, 0, 0), 3)

# 다양한 폰트로 텍스트 그리기
fonts = [
    (cv2.FONT_HERSHEY_SIMPLEX, "SIMPLEX Font"),
    (cv2.FONT_HERSHEY_COMPLEX, "COMPLEX Font"),
    (cv2.FONT_HERSHEY_SCRIPT_SIMPLEX, "SCRIPT Font"),
    (cv2.FONT_HERSHEY_TRIPLEX, "TRIPLEX Font")
]

y_pos = 150
for font, text in fonts:
    cv2.putText(img, text, (50, y_pos), font, 1.2, 
               (255, 0, 0), 2, cv2.LINE_AA)
    y_pos += 80

# 다양한 색상과 크기의 텍스트
colors = [(255, 0, 0), (0, 255, 0), (0, 0, 255)]
sizes = [0.8, 1.0, 1.2]
texts = ["Small Text", "Medium Text", "Large Text"]

for i, (color, size, text) in enumerate(zip(colors, sizes, texts)):
    cv2.putText(img, text, (400, 200 + i * 80), 
               cv2.FONT_HERSHEY_SIMPLEX, size, color, 2)

cv2.imshow('Text Examples', img)
cv2.waitKey(0)
cv2.destroyAllWindows()

    이미지/영상 처리 - 이미지 변환

    8. 이미지 크기 조절 및 회전

    이미지 크기를 조절하고 회전, 반전시키는 기본적인 변환 작업들입니다. 데이터 증강(Data Augmentation)이나 이미지 전처리에서 자주 사용됩니다.

    축소엔 INTER_AREA, 확대엔 INTER_LINEAR/INTER_CUBIC가 무난합니다. 회전 시 중심과 출력 크기를 꼭 확인하세요.

    🔄 주요 변환 함수

    • cv2.resize(): 크기 조절 (보간법 지정 가능)
    • cv2.flip(): 반전 (0:상하, 1:좌우, -1:상하좌우)
    • cv2.getRotationMatrix2D(): 회전 행렬 생성
    • cv2.warpAffine(): 아핀 변환 적용
    import cv2
import numpy as np

# 원본 이미지 로드 또는 테스트 이미지 생성
image = cv2.imread('ex_img/thumbs_up.jpg')
if image is None:
    image = np.zeros((300, 400, 3), dtype=np.uint8)
    cv2.rectangle(image, (50, 50), (350, 250), (0, 255, 0), 3)
    cv2.putText(image, 'TRANSFORM', (80, 160), 
               cv2.FONT_HERSHEY_SIMPLEX, 1.2, (255, 255, 255), 2)

height, width = image.shape[:2]

print(f"원본 이미지 크기: {width} x {height}")

# 1. 이미지 크기 조절 (50% 축소)
resized = cv2.resize(image, (width//2, height//2), 
                    interpolation=cv2.INTER_AREA)

# 2. 이미지 좌우 반전
flipped_h = cv2.flip(image, 1)

# 3. 이미지 상하 반전
flipped_v = cv2.flip(image, 0)

# 4. 이미지 45도 회전
center = (width//2, height//2)
rotation_matrix = cv2.getRotationMatrix2D(center, 45, 1.0)
rotated = cv2.warpAffine(image, rotation_matrix, (width, height))

# 결과 표시
cv2.imshow('Original', image)
cv2.imshow('Resized (50%)', resized)
cv2.imshow('Flipped Horizontal', flipped_h)
cv2.imshow('Rotated 45°', rotated)

cv2.waitKey(0)
cv2.destroyAllWindows()

    이미지/영상 처리 - 필터 및 효과

    9. 이미지 필터와 블러 효과

    다양한 블러 필터를 적용하여 노이즈를 줄이거나 특수 효과를 만들 수 있습니다. 각 필터는 서로 다른 특성을 가지고 있어 목적에 따라 선택해야 합니다.

    커널 크기는 홀수(3,5,7...)를 사용합니다. 미디언은 소금·후추 노이즈에 강하고, 양방향은 테두리를 잘 보존합니다.

    🔍 필터 종류별 특징

    • 가우시안 블러: 가장 자연스러운 블러 효과
    • 평균 블러: 단순하지만 빠른 블러
    • 미디언 블러: 노이즈 제거에 매우 효과적
    • 양방향 필터: 가장자리는 보존하며 노이즈만 제거
    import cv2
import numpy as np

# 원본 이미지 로드 또는 테스트 이미지 생성
image = cv2.imread('ex_img/thumbs_up.jpg')
if image is None:
    # 노이즈가 있는 테스트 패턴 생성
    image = np.zeros((300, 400, 3), dtype=np.uint8)
    for i in range(0, 300, 20):
        cv2.line(image, (0, i), (400, i), (255, 255, 255), 1)
        cv2.line(image, (i, 0), (i, 300), (255, 255, 255), 1)
    cv2.putText(image, 'FILTER TEST', (80, 150), 
               cv2.FONT_HERSHEY_SIMPLEX, 1, (0, 255, 0), 2)

print("=== OpenCV 필터 효과 예제 ===")

# 1. 가우시안 블러 (가장 자연스러운 블러)
gaussian_blur = cv2.GaussianBlur(image, (15, 15), 0)

# 2. 평균 블러 (단순한 블러)
average_blur = cv2.blur(image, (15, 15))

# 3. 미디언 블러 (노이즈 제거에 효과적)
median_blur = cv2.medianBlur(image, 15)

# 4. 양방향 필터 (가장자리 보존하며 스무딩)
bilateral_filter = cv2.bilateralFilter(image, 15, 80, 80)

# 결과 비교 표시
cv2.imshow('Original', image)
cv2.imshow('Gaussian Blur', gaussian_blur)
cv2.imshow('Average Blur', average_blur)
cv2.imshow('Median Blur', median_blur)
cv2.imshow('Bilateral Filter', bilateral_filter)

cv2.waitKey(0)
cv2.destroyAllWindows()

    10. Canny 에지 검출

    Canny 에지 검출은 이미지에서 객체의 윤곽선을 찾는 가장 널리 사용되는 방법입니다. 객체 검출, 윤곽선 추출, 형태 분석의 기초가 됩니다.

    ⚡ Canny 매개변수

    • 낮은 임계값: 약한 에지도 포함 (50~100)
    • 높은 임계값: 강한 에지만 포함 (150~300)
    • 비율: 높은값 = 낮은값 × 2~3이 일반적
    • 전처리로 그레이스케일 변환과 블러링 권장
    import cv2
import numpy as np

# 원본 이미지 로드 또는 테스트 이미지 생성
image = cv2.imread('ex_img/thumbs_up.jpg')
if image is None:
    # 테스트 이미지 생성

    이미지/영상 처리 - 마우스 상호작용

    11. 마우스 이벤트 처리

    OpenCV는 마우스 이벤트를 감지하여 사용자와 상호작용할 수 있습니다. 그림판, 이미지 어노테이션, 관심 영역(ROI) 선택 등에 활용할 수 있습니다.

    마우스 콜백으로 좌표를 받아 도형/영역을 그릴 수 있습니다. 드래그 상태 플래그를 두면 동작이 안정적입니다.

    🖱️ 마우스 이벤트 종류

    • LBUTTONDOWN/UP: 왼쪽 버튼 누름/놓음
    • MOUSEMOVE: 마우스 이동
    • RBUTTONDOWN/UP: 오른쪽 버튼 누름/놓음
    • setMouseCallback()으로 콜백 함수 등록
    import cv2
import numpy as np

print("=== 마우스 상호작용 그림판 ===")
print("마우스 드래그: 그리기")
print("'c' 키: 캔버스 초기화")
print("'q' 키: 종료")

# 전역 변수들
drawing = False  # 마우스 드래그 중인지 확인
ix, iy = -1, -1  # 시작 좌표

# 흰색 캔버스 생성
img = np.ones((400, 600, 3), dtype=np.uint8) * 255

# 마우스 콜백 함수
def mouse_callback(event, x, y, flags, param):
    global ix, iy, drawing, img
    
    if event == cv2.EVENT_LBUTTONDOWN:
        drawing = True
        ix, iy = x, y
        print(f"그리기 시작: ({x}, {y})")
        
    elif event == cv2.EVENT_MOUSEMOVE:
        if drawing:
            # 연속적인 선 그리기
            cv2.line(img, (ix, iy), (x, y), (0, 255, 0), 3)
            ix, iy = x, y
            
    elif event == cv2.EVENT_LBUTTONUP:
        drawing = False
        cv2.line(img, (ix, iy), (x, y), (0, 255, 0), 3)
        print(f"그리기 종료: ({x}, {y})")

# 창 생성 및 마우스 콜백 설정
cv2.namedWindow('Drawing Canvas')
cv2.setMouseCallback('Drawing Canvas', mouse_callback)

while True:
    cv2.imshow('Drawing Canvas', img)
    key = cv2.waitKey(1) & 0xFF
    
    if key == ord('q'):
        break
    elif key == ord('c'):  # 캔버스 초기화
        img = np.ones((400, 600, 3), dtype=np.uint8) * 255
        print("캔버스 초기화됨")

cv2.destroyAllWindows()

    12. 고급 필터 및 마무리

    커스텀 커널을 사용한 컨볼루션 필터와 형태학적 연산을 통해 더욱 전문적인 이미지 처리를 할 수 있습니다. 이것으로 OpenCV 기초를 마무리합니다!

    filter2D의 커널을 바꿔 다양한 효과를 실험해 보세요. 침식/팽창 등 형태학 연산은 마스크 전처리에 특히 유용합니다.

    🚀 더 배워볼 주제들

    • 윤곽선 검출: findContours(), drawContours()
    • 특징점 검출: SIFT, ORB, Harris Corner
    • 객체 추적: Tracking API
    • 머신러닝: HOG, SVM, 딥러닝 연동
    import cv2
import numpy as np

print("=== OpenCV 고급 필터 예제 ===")

# 원본 이미지 생성
image = cv2.imread('ex_img/thumbs_up.jpg')
if image is None:
    image = np.zeros((300, 400, 3), dtype=np.uint8)
    cv2.rectangle(image, (50, 50), (350, 250), (255, 255, 255), -1)
    cv2.putText(image, 'FILTER', (130, 160), 
               cv2.FONT_HERSHEY_SIMPLEX, 1.5, (0, 0, 0), 3)

# 1. 샤프닝 필터 (커스텀 커널)
sharpen_kernel = np.array([[-1, -1, -1],
                          [-1,  9, -1],
                          [-1, -1, -1]])
sharpened = cv2.filter2D(image, -1, sharpen_kernel)

# 2. 엠보싱 필터
emboss_kernel = np.array([[-2, -1,  0],
                         [-1,  1,  1],
                         [ 0,  1,  2]])
embossed = cv2.filter2D(image, -1, emboss_kernel)

# 3. 그레이스케일 + 컬러맵 적용
gray = cv2.cvtColor(image, cv2.COLOR_BGR2GRAY)
colored_map = cv2.applyColorMap(gray, cv2.COLORMAP_JET)

# 4. 형태학적 연산 (침식)
kernel = np.ones((5, 5), np.uint8)
eroded = cv2.erode(image, kernel, iterations=1)

# 결과 표시
cv2.imshow('Original', image)
cv2.imshow('Sharpened', sharpened)
cv2.imshow('Embossed', embossed)
cv2.imshow('Color Map', colored_map)
cv2.imshow('Eroded', eroded)

cv2.waitKey(0)
cv2.destroyAllWindows()
