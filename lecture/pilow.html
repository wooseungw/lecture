1. Pillow 라이브러리

Pillow는 파이썬에서 이미지를 다루는 가장 대표적인 라이브러리입니다. 이미지 열기, 수정, 저장 등 다양한 이미지 처리 작업을 손쉽게 수행할 수 있습니다.

라이브러리 설치

Pillow는 파이썬 표준 라이브러리가 아니므로, 터미널이나 명령 프롬프트에서 `pip`를 사용하여 직접 설치해야 합니다.

# Pillow 라이브러리 설치 명령어
# 터미널(cmd, PowerShell)에 아래와 같이 입력하세요.

pip install Pillow

# ---------------------------------
# 코드에서 Pillow 사용하기
# Image 모듈을 import하여 사용합니다.

from PIL import Image

print("Pillow 라이브러리 준비 완료!")

2. 이미지 열고 정보 확인하기

`Image.open()` 함수로 이미지 파일을 열어 객체로 만듭니다. 생성된 이미지 객체는 형식, 크기, 색상 모드 등 다양한 정보를 속성으로 가지고 있습니다.

이미지 주요 정보

  • format: 이미지 형식 (JPEG, PNG 등)
  • size: 이미지 크기 (가로, 세로 픽셀)
  • mode: 색상 모드 (RGB, L 등)
from PIL import Image

image_path = 'ex_img/victory.jpg'

# 이미지 열기
image = Image.open(image_path)

# 이미지 정보 출력
print(f"형식: {image.format}")
print(f"크기: {image.size}")
print(f"모드: {image.mode}")
print(f"가로: {image.width}px, 세로: {image.height}px")

# 이미지 보기 (새 창으로 열림)
image.show()

3. 이미지 저장 및 형식 변환

`.save()` 메서드를 사용하여 처리된 이미지 객체를 파일로 저장할 수 있습니다. 저장할 파일의 확장자를 다르게 지정하면 이미지 형식이 자동으로 변환됩니다.

형식 변환의 장점

배경을 투명하게 처리하고 싶을 때는 JPG를 PNG로 변환하여 저장할 수 있고, 웹사이트 용량을 줄이기 위해 고화질 이미지를 WEBP 형식으로 변환할 수도 있습니다.

from PIL import Image

image_path = 'ex_img/victory.jpg'
image = Image.open(image_path)

# 새로운 이름으로 이미지 저장 (복사)
output_path = 'ex_img/victory_copy.jpg'
image.save(output_path)
print(f"이미지 복사 완료: {output_path}")

# PNG 형식으로 변환하여 저장
png_path = 'ex_img/victory.png'
image.save(png_path)
print(f"PNG 형식으로 저장 완료: {png_path}")

4. 이미지 크기 조절

`.resize((가로, 세로))` 메서드를 사용하여 이미지의 크기를 원하는 대로 변경할 수 있습니다. 썸네일 제작이나 이미지 규격 통일에 유용합니다.

`.thumbnail()` 메서드

`.thumbnail()`은 원본 비율을 유지하면서 주어진 크기 안에 들어가는 최대 썸네일을 만듭니다. 원본 객체를 직접 수정하므로 복사본을 만들어 사용하는 것이 안전합니다.

from PIL import Image

image = Image.open('ex_img/victory.jpg')
width, height = image.size

# 1. 이미지 크기 반으로 줄이기
small_image = image.resize((width // 2, height // 2))
small_image.save('ex_img/victory_small.jpg')
print(f"작은 이미지 저장: {small_image.size}")

# 2. 정사각형으로 만들기 (200x200)
square_image = image.resize((200, 200))
square_image.save('ex_img/victory_square.jpg')
print(f"정사각형 이미지 저장: {square_image.size}")

5. 자르기, 회전, 반전

이미지의 특정 부분만 잘라내거나, 이미지를 회전시키고, 좌우 또는 상하로 뒤집는 등 다양한 기하학적 변형을 적용할 수 있습니다.

자르기(Crop) 좌표계

`.crop((left, top, right, bottom))` 메서드는 왼쪽 위 모서리를 (0,0) 기준으로 하여 잘라낼 사각형 영역의 좌표를 받습니다.

from PIL import Image

image = Image.open('ex_img/victory.jpg')
width, height = image.size

# 1. 이미지 중앙 부분 자르기
box = (width//4, height//4, width*3//4, height*3//4)
cropped_image = image.crop(box)
cropped_image.save('ex_img/victory_cropped.jpg')

# 2. 이미지 90도 회전
rotated_90 = image.rotate(90, expand=True)
rotated_90.save('ex_img/victory_rotated.jpg')

# 3. 이미지 좌우 반전
flipped_h = image.transpose(Image.FLIP_LEFT_RIGHT)
flipped_h.save('ex_img/victory_flipped.jpg')

6. 색상 모드 변환

`.convert()` 메서드를 사용하여 이미지의 색상 모드를 변경할 수 있습니다. 컬러 이미지를 흑백으로 바꾸거나, 특정 모드로 변환하여 처리 효율을 높일 수 있습니다.

주요 색상 모드

  • 'RGB': 3채널 컬러 이미지 (빨강, 초록, 파랑)
  • 'L': 1채널 그레이스케일 (밝기 정보만)
  • '1': 1비트 흑백 이미지 (검정 또는 흰색)
  • 'RGBA': RGB에 투명도(Alpha) 채널 추가
from PIL import Image

image = Image.open('ex_img/victory.jpg')

# 1. 그레이스케일로 변환
grayscale = image.convert('L')
grayscale.save('ex_img/victory_grayscale.jpg')
print(f"그레이스케일 저장 완료: {grayscale.mode}")

# 2. 흑백 (1비트) 이미지로 변환
black_white = image.convert('1')
black_white.save('ex_img/victory_black_white.jpg')
print(f"흑백 이미지 저장 완료: {black_white.mode}")
